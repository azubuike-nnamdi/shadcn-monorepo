# Colors
#### Application of the color palette brings a unified and recognizable consistency to MTNâ€™s array of digital products and interfaces. This consistency is grounded in a set of well-defined rules about how to work with the Sunshine component library in the context of dark and light themes.
### Color Anatomy
#### The neutral gray family is dominant in the default themes, making use of subtle shifts in value to organize content into distinct zones. The core yellow family serves as the primary action color across all MTN products and experiences. Additional colors are used sparingly and purposefully.
